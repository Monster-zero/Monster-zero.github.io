搭建前的准备

一、下载安装Node.jsh和gii

二、注册github（传说这是一个程序员间的同性交友平台）如果有的话可以

三、配置Node.js和git以及配置github

步骤1、安装Node.js和git

这里两个的安装都很简单如果是在Windows操作系统下都只需要“下一步”就安好了

安装好后在cmd控制台输入以下命令查看版本号（有版本号就说明安装成功）

git命令也是同上

步骤2、在github上创建仓库

找到greate a new repository点击创建格式如下

再点击选择设置settings

查看你的网址这个时候你就可以在任意的浏览器上访问这个网址都可以访问到你的github界面

步骤3、配置git 和安装hexo

首先要在你的一个本地磁盘里建一个文件夹以放置博客文件

如果你安装了上面说的git进入到blog文件夹右击有个gti bash here点击它

首先要设施名字和邮箱都是随便填的

然后设置ssh秘钥用于连接github

输入cd ~/.ssh，检查是否由.ssh的文件夹(如果没有就输ssh-keygen -t rsa)

输入命令 cat id_ras.pub 把里面的内容复制

登陆github 登录点击头像下的settings，添加ssh

把刚才的秘钥粘贴进去

输入ssh -T git@github.com，测试添加ssh是否成功。如果看到Hi后面是你的用户名，就说明成功了

安装hexo

安装Hexo，在自己认为合适的地方创个文件夹，我是在D盘建了一个blog文件夹。然后通过命令行进入到该文件夹里面

输入npm install hexo -g，开始安装Hexo

输入hexo -v，检查hexo是否安装成功（有版本号就是成功了）

输入hexo init，初始化该文件夹（出现下面情况就是成了）

输入npm install，安装所需要的组件

安装好后就可以部署了 hexo g

部署好后输入hexo server 有一个本地网址你可以在浏览器访问查看你的博客

安装好后进入到你的blog文件在其文件夹中，找到_config.yml文件，修改repo值（在末尾）

repository的值是你的github上面的连接方式入下图

然后你再hexo g从新部署一便

使用hexo d可以上传到github上这个时候你再访你的那个github网址就可以从网上看到你部署的博客了
ps:这个时候你就可以把你的链接发给你的小伙伴了
